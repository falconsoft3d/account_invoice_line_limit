Maximum amount of lines in invoice

Allows you to configure the number of lines in an invoice.

Setting

Setting -> Company

The maximum limit of lines desired must be configured in company.

Limit for sale and Purchase

This does not it will prevent more lines from being added, it will only alert the user that it has exceeded the desired limit. There we indicate the number of desired lines, then we save so that they apply the changes.

Invoice Creation

Gives a message showing the line limit on the invoice
